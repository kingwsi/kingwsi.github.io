java8特性之流式处理

java8使用流操作（Stream）来优雅的处理数据

Stream是什么？

Stream 作为 Java 8 的一大亮点，它与 java.io 包里的 InputStream 和 OutputStream 是完全不同的概念。Stream是针对集合对象的增强，Stream的出现使得对集合的操作变得更加优雅，尤其是对大量数据进行处理的时候。

如何使用

Stream 就如同一个迭代器（Iterator），单向，不可往复，数据只能遍历一次，遍历过一次后即用尽了，就好比流水从面前流过，一去不复返。
