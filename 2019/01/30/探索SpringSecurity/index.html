探索SpringSecurity

SpringSecurity是什么，如何使用

SpringSecurity 是什么？

SprinSecurity是一个基于Spring的安全框架，功能强大且高度可自定义，提供了强大的身份验证和授权功能

运行一个最简单的security认证

在SpringBoot中使用SpringSecurity
创建SpringBoot工程,引入Security依赖

开启security验证

运行项目，访问localhost:8090，可以看到 会弹出httpBasic验证

查看项目运行日志，可以看到项目启动是SpringSecurity为我们自动配置了密码Using default security password: 1e0e565a-22d1-45a7-8037-774ab8702379，使用用户名user登录即可

为SpringSecurity配置更多验证逻辑

继承WebSecurityConfigurerAdapter类，并重写其configure(HttpSecurity http)方法

这是个验证链，可通过对其配置实现更多需求，如登录页面跳转.formLogin("/path")

SpringSecurity运行流程是什么？

客户端请求过来之后，会经历那些步骤

在浏览器请求localhost:8090/hello时，我们配置了security的过滤器，过滤器会检查请求中有没有要处理的请求，比如登录过滤器，他会处理post方式的/login请求，其他则不处理，所以/hello请求会直接到绿色部分，检查过滤器链中没有通过登录过滤器，然后由红色部分处理，并重定向到登录页面
